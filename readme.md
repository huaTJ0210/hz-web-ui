### 1、项目介绍

> 一个基于 pnpm 的 monorepo 项目, 使用 vue3 + TS 构建业务基础组件库、工具库的模板；

### 2、项目使用简介

#### 2.1、安装依赖

> pnpm install

#### 2.2、创建组件

> pnpm run c

#### 2.3 开发中调试组件

> - 在 app/src/App.vue 中 使用新创建的组件
> - pnpm run test

#### 2.5 **组件、工具包的发布**

> 项目全局搜索@htj24 替换为自己的组织名 （** pnpm install 前进行替换 **）

### 3、配置文件

#### 3.1 ./config.ts

> 主要配置 组件名称 和 组件库名称
